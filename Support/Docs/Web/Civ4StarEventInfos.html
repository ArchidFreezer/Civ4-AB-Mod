Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
BadTextKeyGame text to display for negative star events that impact a city
GoodTextKeyGame text to display for positive star events that impact a city
MitigateTextKeyGame text to display when a negative star event that would have impacted a city is negated
NeutralTextKeyGame text to display when a star sign does not impact a city
TargetTypeNO_TARGETReference to the StarEventTargetType that the event will target
TypeUnique reference to the star event

Integer

TagDefaultPurpose
iAggregateResultsCap0If greater than this number of cities is impacted by the event then only one screen message is shown rather than one per city
iBadEffectModifier0Percentile modifier to the event strength if it has a negative effect
iCombatPoints0Change to the number of great general points a player for each city impacted by the event
iConscript0Number of conscript units a city gains if impacted by this event
iCulture0Change in culture of a city impacted by this event
iFood0Change in food stored in a city impacted by this event
iGold0Change to a players gold for each city impacted by the event
iGreatPersonPoints0Change in great person points stored in a city impacted by this event
iHappyTurns0Number of turns of happiness for a city impacted by this event
iMissionary0Number of missionaries of the players state religion that may be created in each city impacted. If the city does not have the state religion it will be applied instead.
iNoEffectChance0Percentile chance that the event will have no impact
iNumTargets0Maximum number of cities the event will impact
iPersistDecayRate0The rate at which a persistent event decays, so that its effects lessen over time until it ceases to have an effect. The higher the number here the quicker the event will decay
iPopulationChange0Change in population of a city impacted by this event
iProduction0Change in production points (hammers) of a city impacted by this event
iScalePoints0Percentile change in the effect of the event, this is applied to all other city and player modifiers but will never increase the effect beyond the stated maximums

Boolean

TagDefaultPurpose
bPersistent0If true, the event may continue to have an impact over multiple turns
