Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
DescriptionA reference to the text files, which contain an entry for this world size
HelpAdditional help text that will be displayed in the Platypedia
TypeThe internal name of this world size

Integer

TagDefaultPurpose
iAdvancedStartPointsMod0Percentage change to the number of advanced start points a player receives when starting a new advanced start game
iBuildingClassPrereqModifier0Percentage change to the number of prereq building classes required to construct a building
iColonyMaintenancePercent0Percentage change to the maintenance cost of a city based on the number of colonies (cities in different areas from the capital). This value modifies the number of citries taken into account when working out the maintenance cost.
iCorporationMaintenancePercent0Percentage change to the maintenance cost of a city based on the number of corporations it has present
iDefaultPlayers0The default number of players when a new game is selected using this world size
iDistanceMaintenancePercent0Percentage change to the maintenance cost of a city based on the distance from the closest government centre (capital or building with bGovernmentCenter set to true)
iFeatureGrainChange0Used by the Python MapGenerator, it impacts how big clusters of features are. The smaller the value the larger the relative cluster sizes of feature types
iGridHeight0The height of the world map in tiles
iGridWidth0The width of the world map in tiles
iMaxConscriptModifier0Percentage change to the number of units that can be conscripted in a city each turn
iNumCitiesAnarchyPercent0Percentage change to the duration of anarchy suffered by a player based on the number of cities they have
iNumCitiesMaintenancePercent0Percentage change to the maintenance cost of a city based on the total number of cities the player has
iNumFreeBuildingBonuses0The number of free bonuses that a building provides if the BuildingInfo iNumFreeBonuses is set to -1
iResearchPercent0Percentage change to the cost of researching techs. It is also used as a percentage change to the cost of espionage missions
iTargetNumCities0This is the number of cities the AI works towards owning in a world of this size. It is used in various AI decisions such as choosing city production; the value of buildings, tech, etc.; starting locations ...
iTerrainGrainChange0Used by the Python MapGenerator, it impacts how big clusters of terrains are. The smaller the value the larger the relative cluster sizes of terrain types
iTradeProfitPercent0Percentage change in the profit a city gains from trade
iUnitNameModifier0Unknown. This value is not used in the DLL and not exposed to the EXE
iWarWearinessModifier0Percentage change applied to the war weariness in each city
