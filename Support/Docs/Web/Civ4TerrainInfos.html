Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
ArtDefineTagReference to the art definition for the terrain
ButtonPath to a .dds file that contains the image for the object
CivilopediaThe pedia text for the terrain
DescriptionReference to the text that contains a description of the terrain. The text may be defined in any of the files in \Assets\XML\Text
HelpAdditional help text that will be displayed in the Pedia
TypeThe internal reference to the terrain
WorldSoundscapeAudioScriptSound played on close zooms for the terrain

Integer

TagDefaultPurpose
iBuildModifier0Percentage change to the cost of improvements on this terrain
iDefense0Percentage change to the defdensive bonus of troops on the terrain
iMovement1Number of movement points required to move through the terrain
iSeeFrom0The level (height) of the terrain, used when determining which plots are visible from the plot
iSeeThrough0The level (height) of the terrain, used when determining whether units can see plots behind the one this terrain is on

Boolean

TagDefaultPurpose
bFound0If true, it is possible to found a city on the terrain
bFoundCoast0If true, it is possible to found a city on this terrain if it is next to a coast line
bFoundFreshWater0If true, it is possible to found a city on this terrain if it is next to a fresh water supply
bGraphicalOnly0If true, the terrain is not a true terrain, but rather a way of displaying the tile. Used for hills and peaks, which are plot types rather than true terrains.
bImpassable0If true, the tearrin is considered impassable to normal non-air units.
bWater0If true, the terrain is considered to be water for the purposes of determining which domain of unit can pass through it, etc.

Compound (Multi-line)

TagDefaultPurpose
