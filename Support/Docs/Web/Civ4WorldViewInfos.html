Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
DescriptionThe game text to display for the world view
EnactButtonThe button used when enacting the world view
EnactTextKeyThe game text displayed when asking if the player wishes to enact the world view
RepealButtonThe button used when repealing the world view
RepealTextKeyThe game text displayed when asking if the player wishes to repeal the world view
TypeThe unique identifier for the world view
