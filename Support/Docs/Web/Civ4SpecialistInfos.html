Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
ButtonPath to a .dds file that contains the image for the object
CivilopediaText to display in the pedia
DescriptionReference to the text that contains the display name of this specialist. The text may be defined in any of the files in \Assets\XML\Text
GreatPeopleUnitClassNO_UNITCLASSIf the specialist generates Great Person Points, this value specifies for what Great Person unit type it does so.
HelpAdditional help text that will be displayed in the Pedia
HotKeyThe key, that when pressed selects this object
StrategyThis tag links to an entry in the text files which describes to the player how to use this specialist.
TextureGives the directory path to the texture file used for this specialist. The default images are located in \Assets\Art\Interface\MainScreen\CityScreen\.
TypeThe internal reference to this specialist

Integer

TagDefaultPurpose
iExperience0Gives units built in this city an additional amount of experience points
iGreatPeopleRateChange0If GreatPeopleUnitClass is not NO_UNITCLASS (that is, the specialist generates Great People Points), this sets the number of GPP per turn
iHappinessChange0Change to city happiness
iHotKeyPriority0If two objects have similar HotKey combinations then the one with the highest priority will take precedence
iOrderPriority0The priority of this specialist when determining the order in which actions are performed. A specialist with a high order priority will act first
iStarSignMitigateChangePercent0Percentile change in the chance to mitigate the effects of a negative star event
iStarSignScaleChangePercent0Percentile change to the effect that a star event has on an impacted city

Boolean

TagDefaultPurpose
bAltDown0If true, the HotKey is only used if the Alt key is also pressed
bCtrlDown0If true, the HotKey is only used if the Control key is also pressed
bShiftDown0If true, the HotKey is only used if the Shift key is also pressed
bSlave0If true, the specialist is considered a slave unit
bVisible0If true, the specialist is displayed where the default Priest, Engineer, etc. specialists are located in the game. If false, the specialist is located at the bottom right of the city screen, where the default settled Great People are situated.
