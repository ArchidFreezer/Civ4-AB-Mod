Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
CityCultureSets the minimum CultureLevel a city needs to attain to count towards a cultural victory
CivilopediaPedia text for this victory type
DescriptionReference to the text that contains the name of this victory type. The text may be defined in any of the files in \Assets\XML\Text
TypeThe internal reference this victory type
VictoryMovieNO_MOVIEReference to a movie file that is played when a player acheives this victory

Integer

TagDefaultPurpose
iLandPercent0Total percentage of land you and your vassals must control to achieve this victory
iMinLandPercent0Minimum percentage of land that you personally must control to achieve victory.
iNumCultureCities0Number of cities that must meet the <CityCulture> tag's level of culture points to acheive this victory.
iPopulationPercentLead0The percentage of your teams population against the world population, in addition to being the most populous team that must be reached to achieve this victory. The actrual formula has more factors that this simple explanation.
iReligionPercent0Percentage of cities that must be following your state religion (when you control the holy city) in order to win.
iTotalCultureRatio0You must have at least this percentage of the highest players culture value to achieve the victory. It is a gate to ensure that a player cannopt be too far behind in the culture race.
iVictoryDelayTurns0Number of turns before the victory is registered officially by the game and a win is triggered. This is used for the Spaceship victory condition.

Boolean

TagDefaultPurpose
bConquest0If true, this victory requires the elimination of all enemies and qualifies as a Conquest win
bDiploVote0If true, this victory requires the player to be elected in a worldwide vote and qualifies as a Diplomatic win
bEndScore0If true, the victory is based on the player with the highest score at the end of the game (2050 by default)
bPermanent0Unused in the dll, exposed to the exe and Python
bTargetScore0If true, the victory will trigger if a team reaches a target score as defined in a wouldbuildersave file.
