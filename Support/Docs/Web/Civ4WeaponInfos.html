Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
ButtonPath to the button to display for units with this weapon type
CivilopediaThe civilopedia description of the weapon
DescriptionTye in game description of the weapon
TypeThe unique identifier of the weapon

Integer

TagDefaultPurpose
iStrength0The additional strength that a unit has with this weapon
iTier0The tier of weapon. Unit may be restricted to lower level tiers.

Boolean

TagDefaultPurpose
bAmmunition0If true, this is a projectile
