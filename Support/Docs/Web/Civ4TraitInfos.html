Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
DescriptionA reference to the text files, which contain an entry for this trait. This is only required for the ParentTrait of multi-level traits
FoundCityCultureLevelThe minimum culture level that a city founded by a player with this trait will have
HelpAdditional help text that will be displayed in the Platypedia
ShortDescriptionA reference to the text files, which contains the short version which is typically a 3 character abbreviation. This is only used for LeaderHead traits, but should be considered mandatory for those.
TypeThe internal name of this Trait

TagDefaultPurpose

iFoundCityPopulationChange0The minimum population that a city will have on founding
iGoldPercentDividendPerTurn0Percentage change in the gold return for a player

iGreatPeopleRateModifier0Change to the rate that Great Person points are gained in each city
iHappiness0The extra happiness each city gains
iHealth0The extra health each city gains
iLevelExperienceModifier0Percentage change in required experience points per promotion
iMaxAnarchy-1Maximum number of turns of anarchy. -1 = no limit; 0 = no anarchy
iMaxCivicAnarchyTurns0If the value is > 0 then this is the maximum number of anarchy during Civic changes
iMaxGlobalBuildingProductionModifier0Percentage change in the production of all world wonders in your cities
iMaxPlayerBuildingProductionModifier0Percentage change in the production of all national wonders in your cities
iMaxReligionAnarchyTurns0If the value is > 0 then this is the maximum number of anarchy during State Religion changes
iMaxTeamBuildingProductionModifier0Percentage change in the production of all team wonders in your cities
iMissionarySurvivalChance0Percentage chance that a Missionary will survive a spread religion attempt
iOccupationTimeChange0Additional change in occupation timer each turn
iSettlerReligionLevel0If 1, then a settler will spread the state religion to a founded city. If 2, then a temple will also be constructed
iStarSignMitigateChangePercent0A percentile multiplier that impacts the scale of any star sign applied to the players cities
iStarSignScaleChangePercent0A percentile multiplier that impacts the scale of any star sign applied to cities
iUnitRangeChange0A change in range applied to all units with range restrictions
iUnitRangeModifier0Percentage change in the range of units that have a range restriction
iUnitWithdrawalHealRate0A unit that withdraws form combat will have this percentage of its damage healed
iUpkeepModifier0Percentage amount civic maintenance costs are reduced
iWarWearinessModifier0Percentage change in War Weariness suffered by the player

Boolean

TagDefaultPurpose
bUnitRangeUnbound0If true, units have all range restrictions, except UNITRANGE_HOME, lifted
bUnitTerritoryUnbound0If true, units that have a range of UNITRANGE_TERRITORY have the restriction lifted and can move outside the civ territory
