Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
BonusProducedNO_BONUSThe bonus type produced by the corporation, if any
ButtonPath to the button to display for the Corporation
CivilopediaRefers to a tag in the Civ4GameText file which stores the Platypedia description for the Corporation
DescriptionA reference to the text files, which contain an entry describing this Corporation
FreeUnitClassNO_UNITCLASSThe first to found this Corporation receives a number of units of this class, see iFreeUnits
MovieFilePath to the movie file to be played when the Corporation is founded
MovieSoundReference to the sound file used when playing the movie
SoundReference to the sound file used when the Corporation spreads to a new city
TechPrereqNO_TECHThe tech that unlocks this religion
TypeThe internal name of this Corporation

Integer

TagDefaultPurpose
iMaintenance0The maintenance cost for this corporation is each city that has it. This value is multiplied by the number of corporation prereq bonuses the city has access to
iOrderPriority0Has no meaning for Corporations
iSpreadCost0The amount it costs to spread this corporation to a new city
iSpreadFactor0Determines how likely the Corporation is to spread to other cities without missionaries or buildings
iTGAIndexThe index into the TGA file for the corporation icon.

Compound (Multi-line)

TagDefaultPurpose
CommercesProducedNONEThe commerce gained in a city hosting this Corporation per resource consumed. The units are percentages so a value of 25 would require 4 resources to provide a single hammer/beaker, etc. This value is also impacted by the world size corporation percentage value.
  <iCommerceProduced>0</iCommerceProduced>
HeadquarterCommercesNONEThe commerce gained in the city hosting the headquarters of this Corporation
  <iHeadquarterCommerce>0</iHeadquarterCommerce>
PrereqBonusesNONEThe bonuses consumed by this corporation, one of which must be present in the city to spread the corporation there and for the city to gain any benefit
  <BonusType></BonusType>
YieldsProducedNONEThis defines the extra yields that the corporation provides, per resource the corporation consumes, using the iYieldProduced integer tag. iYieldProduced controls the extra changes in food, hammer, and commerce production in each city with the corporation present. (Ex. 100 = one extra Food, Hammer, Gold. 25 = .25 extra Food, Hammer, Gold) 
  <iYieldProduced>0</iYieldProduced>
