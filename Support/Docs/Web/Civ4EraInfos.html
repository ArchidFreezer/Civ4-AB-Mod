Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
AudioUnitDefeatScriptNO_AS2DSound used when unit loses combat
AudioUnitVictoryScriptNO_AS2DSound used when unit wins combat
DescriptionA reference to the text files, which contain an entry for this Era
StrategyRefers to a tag in the Civ4GameText file which stores the Ppedia strategy for the Era
TypeThe internal name of this Era

Integer

TagDefaultPurpose
iAdvancedStartPoints0Number of advanced start points given
iAnarchyPercent0Percentage change in the length of periods of anarchy
iBuildPercent0Percentage change in the work rate of units

iCulturePercent0Percentage change in the threshold for city culture growth
iEventChancePerTurn0Multiplier applied to the chance of events happening each turn
iFreePopulation0Amount of extra population points to cities founded for this era start
iGreatPeoplePercent0Percentage change in the threshold for creating great people
iGrowthPercent0Percentage change to the rate that cities and other game elements grow at
iImprovementPercent0Percentage change in the time taken for an improvement to upgrade
iMaxCities0The maximum number of cities a player can control in this era
iResearchPercent0Percentage change in the research cost of techs
iSoundtrackSpace0Unknown, not used in the DLL, exported for the EXE
iStartPercent0Percentage of the expected number of game turns that starts in this era begin at
iStartingDefenseUnits0Number of defensive units given to a player at the start of the game
iStartingExploreUnits0Number of explorer units given to a player at the start of the game
iStartingGold0Amount of gold given to each player at the start of the game
iStartingUnitMultiplier0Multiplier applied to the number of starting units a player has. Used for late starts
iStartingWorkerUnits0Number of worker units given to a player at the start of the game
iTechCostModifier0Multiplier applied to the rate at which research points are gained
iTrainPercent0Percentage change in the production cost of units
iUnitRangeChange0Absolute change in range for a range bound unit
iUnitRangeModifier0Percentile change in range for a range bound unit

Boolean

TagDefaultPurpose
bFirstSoundtrackFirst0Unknown, not used in the DLL, exported for the EXE
bNoAnimals0If true, then no animals will be spawned in this era
bNoBarbCities0If true, no barbarian cities will be created
bNoBarbUnits0If true, n barbarian units will be spawned
bNoGoodies0If true, then any map created using this era will have no goody huts placed on it
bUnitRangeUnbound0If true, units with a maximum range have the range cap removed
bUnitTerritoryUnbound0If true, units that are normally restricted ot the players territory may venture outside it
