Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
AdvisorDefines the type of advisor that would recommend this building in the in-game menus
ArtDefineTagRefers to a tag in the Civ4ArtDefines_Building file
BuildingClassThe class of the building, as defined in the Civ4BuildingClassInfos file
CivicOptionNO_CIVICOPTIONAll the civics in this civic option are available to the player on construction
CivilopediaRefers to a tag in the Civ4GameText file which stores the Pedia description for the building
ConstructSoundNO_SOUNDThe sound to play when the building is constructed
CreateFeatureTypeNO_FEATUREA feature that will be created on the city plot on construction of the building
DescriptionA reference to the text files, which contain an entry for this Building
DiploVoteTypeNO_DIPLOVOTEThe type of diplomatic vote that this building allows the player to participate in
FoundsCorporationNO_CORPORATIONIf set, the corporation that will be create din the city if this building is constructed
FreeBonusNO_BONUSMakes this bonus available to the city, the number of free bonuses is defined in iNumFreeBonuses
FreeBuildingNO_BUILDINGProvides this building free to every city
FreeStartEraNO_ERABuilding is free if you start in this Era or later.
FreeUnitClassNO_UNITCLASSOn construction of this building a unit of this unit class will be created in the city
GlobalCorporationCommerceNO_CORPORATIONEarns gold from every city with this corporation.
GlobalReligionCommerceNO_RELIGIONEarns gold from every city with this religion.
GreatPeopleUnitClassNO_UNITCLASSType of Great Person points the building generates
HelpAdditional help text that will be displayed in the Pedia
HolyCityNO_RELIGIONIf set, the building can only be constructed in the Holy City of the religion specified
HotKeyThe key, that when pressed selects the building.
MaxStartEraNO_ERAIf the game is started in an ERA after this then the building is not available.
MinCultureLevelNO_CULTURELEVELThe minimum culture level that the city must be at in order to construct this building
MovieDefineTagSpecifies the tag in the Civ4ArtDefines_Movie file that will play when this building is constructed
NoBonusNO_BONUSBlocks this bonus from being available in the city
ObsoleteTechNO_TECHTech that makes the building obsolete (like Economics for Castles).
PowerBonusNO_BONUSProvides power to the city if this bonus is available.
PrereqBonusNO_BONUSBonus required to construct this building
PrereqCorporationNO_CORPORATIONIf set, the Building can only be constructed if the specified corporation is present in the city.
PrereqReligionNO_RELIGIONIf set, the Building can only be constructed if the specified Religion type is present in the city.
ReligionTypeNO_RELIGIONIf set, the Building is considered to be on the specified religion when determining things such as religious building happiness, etc.
SpecialBuildingTypeNO_SPECIALBUILDINGDetermines whether or not the building is one the special types listed in the Civ4SpecialBuildingInfos file, like a monastery or cathedral
StateReligionNO_RELIGIONIf set, the Building can only be constructed while the specified Religion type is the Civilizations State Religion
StrategyRefers to a tag in the Civ4GameText_Strategy file that stores the strategic or usage info on the Building, displayed in mouseover pop-ups
TypeThe internal name of this Building
VictoryPrereqNO_VICTORYVictory type required for construction of the building (e.g. the Diplomacy victory condition must be enabled for the United Nations to be available)
