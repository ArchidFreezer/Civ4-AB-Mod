Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
ButtonPath to a .dds file that contains the image for the build
DescriptionReference to the text that contains the name of the build. The text may be defined in any of the files in \Assets\XML\Text
EntityEventThis describes the graphics displayed while this build is being constructed (e.g. a worker using a shovel).
HelpAdditional help text that will be displayed in the Pedia
HotKeyThe key, that when pressed selects this object
ImprovementTypeNO_IMPROVEMENTReference to an improvement created on completion
ObsoleteTechNO_TECHA technology, that once researched, prevents this build from being available for a players units
PrereqTechNO_TECHThe technology required to initiate this build type
RouteTypeNO_ROUTEReference to a route created on completion
TypeThe internal reference to this build

Integer

TagDefaultPurpose
iCost0The cost in gold to initiate the build
iHotKeyPriority0If two objects have similar HotKey combinations then the one with the highest priority will take precedence
iOrderPriority0The priority of this build when determining the order in which actions are performed. A build with a high order priority will be processed first
iTime0The base time taken to complete the build. The actual number of turns is dependant on the work rate of the units performing the build.

Boolean

TagDefaultPurpose
bAltDown0If true, the HotKey is only used if the Alt key is also pressed
bCtrlDown0If true, the HotKey is only used if the Control key is also pressed
bKill0If true, the unit completing the build will be killed
bShiftDown0If true, the HotKey is only used if the Shift key is also pressed

Compound (Multi-line)

TagDefaultPurpose
FeatureStructsNONEAny features that impact the time te build takes, any hammers gained on completion of the build, or will be removed on completion of the build
