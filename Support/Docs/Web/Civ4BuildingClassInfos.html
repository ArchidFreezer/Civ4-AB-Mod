Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
DefaultBuildingNO_BUILDINGThe building used when constructing a building of this class if the civ does not have a unique bulding
DescriptionReference to the name of this building class. The text may be defined in any of the files in \Assets\XML\Text
TypeThe internal reference to this building class

Integer

TagDefaultPurpose
iCategoryNAUsed by the java tooling to order the infos. Not read by the dll.
iExtraPlayerInstances0Number of building class that can be constructed if the limit has already been reached (used to allow the Palace to be moved)
iMaxGlobalInstances-1Number of building class that can exist in the world; -1 means no limit
iMaxPlayerInstances-1Number of building class that can exist for a player; -1 means no limit
iMaxTeamInstances-1Number of building class that can exist in one team; -1 means no limit

Boolean

TagDefaultPurpose
bMonument0Unused, exposed to Python
bNoLimit0If true, there is no limit on the number that can be constructed in the same city
