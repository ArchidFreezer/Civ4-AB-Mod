Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
ButtonThe button to display the promotion
DescriptionA reference to the text files, which contain an entry for this promotion
HelpAdditional help text that will be displayed in the Platypedia
HotKeyThe key, that when pressed selects the promotion.
LayerAnimationPathUnknown; used within the exe
ObsoleteTechOnce this tech has been researched the promotion will no longer be available for units to take.
PromotionPrereqNO_PROMOTIONA unit must have the promotion defined in order to gain this one
SoundSound played when the promotion is given to a unit; typically AS2D_IF_LEVELUP
StateReligionPrereqNO_RELIGIONThe player must have this religion as their state religion in order for their units to gain this promotion
TechPrereqNO_TECHThe player must know the tech before their units can gain this promotion
TypeThe internal name of this promotion

Boolean

TagDefaultPurpose
bAltDown0If true, the HotKey is only used if the Alt key is also pressed
bAlwaysHeal0If true, the recipient can heal whilst moving
bAmphib0If true, the recipient can attack from a transport without penalty
bBlitz0If true, the recipient can attack multiple times per turn
bCanMovePeaks0If true, the unit can pass over peaks
bCarryReligion0If true, a settler that has this promotion will automatically spread the players state religion to any city it founds. Has no effect for players with no state religion.
bCityPrereq0If true, this promotion can only be gained when a unit is in a city plot
bCtrlDown0If true, the HotKey is only used if the Control key is also pressed
bEnemyRoute0If true, recipient can use enemy roads as if they were friendly
bHillsDoubleMove0If true, the recipient can move twice as far in hills
bImmuneToFirstStrikes0If true, the recipient is immune to First Strikes
bLeader0If true, the recipient is considered a warlord
bLoyal0If true, spy units will never reveal their civ.
bRiver0If true, the recipient can attack across a river without penalty
bShiftDown0If true, the HotKey is only used if the Shift key is also pressed
bSpyRadiation0If true, spies will be able to use nuclear devices in missions
bUnitRangeUnbound0If true, the unit has no range cap
bUnitTerritoryUnbound0If true, the unit can move out the the players territory
