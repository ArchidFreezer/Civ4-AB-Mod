Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
ButtonPath to a .dds file that contains the button on the city screen
DescriptionReference to the text that containms a description of an instance of this object. The text may be defined in any of the files in \Assets\XML\Text
StrategyText that appears in the pedia describing some strategy hints for using this emphasis
TypeThe internal reference to an object instance of this type

Boolean

TagDefaultPurpose
bAvoidGrowth0If true, the city is managed so that it does not grow in size
bGreatPeople0If true, the governor focuses on production of Great People Points.
