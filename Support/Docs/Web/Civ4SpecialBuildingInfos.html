Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
ButtonPath to a .dds file that contains the image for the special building type
DescriptionReference to the text that contains a description of this special building. The text may be defined in any of the files in \Assets\XML\Text
ObsoleteTechNO_TECHTechnology that obsoletes all building with this special building type
TechPrereqNO_TECHTechnology required to be known by a player to construct all building with this special building type
TechPrereqAnyoneNO_TECHTechnology, which when acquired by any player allows the construction of all building with this special building type
TypeThe internal reference to this special building type

Boolean

TagDefaultPurpose
bValid0If false, the special building cannot be built globally until something unlocks it

Compound (Multi-line)
