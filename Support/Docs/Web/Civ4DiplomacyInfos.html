ResponsesNONESet of diplomacy responses. contains a set of tags which define when the response may be used. These are the set of civilizations that use the response, the leaders that use the response, the attitude level the response can be used at, whether the response is used when the AI is stronger or weaker than the other civ and the reference to the actual text of the response.
