Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
CivicNO_CIVICCivic the player must be running in order to participate in the vote
DescriptionReference to the text that contains the name of the vote source. The text may be defined in any of the files in \Assets\XML\Text
FreeSpecialistNO_SPECIALISTFree specialist given to members of the vote source
PopupTextText displayed when choosing a vote
SecretaryGeneralTextText displayed for the leader (secretary general) of the vote source
TypeThe internal reference to this vote source
