Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
DescriptionThe description of the effect, this is not displayed in the game
PathThe path to the effect graphics files
TypeThe unique identifier of the effect

Integer

TagDefaultPurpose
bIsProjectile0If true, the effect is considered a projectile

Boolean

TagDefaultPurpose
bSticky0This is only set in the OOB effects for air effects, it is unused in the DLL

Float

TagDefaultPurpose
fArcValue0.0If the effect is considered a projectile this value determines the arc at which the projectile is launched
fScale0.0The scale applied to the graphics file when used in game
fSpeed0.0For peojectiles, the speed at which the projectile travels
fUpdateRate0.0Unknown, assumed to be the frequency that the effect graphic is updated; unused in the DLL
