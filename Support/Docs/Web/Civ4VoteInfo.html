Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
DescriptionReference to the text that contains the name of this vote type. The text may be defined in any of the files in \Assets\XML\Text
TypeThe internal reference to this vote type

Integer

TagDefaultPurpose
iMinVoters0Number of voting members necessary for the vote to go ahead
iPopulationThreshold0Percentage change to the number of votes that are required for a vote to go ahead. If the <bCityVoting> or <bCivVoting> flags are set then it relates to those values and not the population.
iStateReligionVotePercent0Percentage change to the number of votes a player has in religious votes if the religion is their state religion.
iTradeRoutes0Number of trade routes granted to each member if vote passes

Boolean

TagDefaultPurpose
bAssignCity0Whether vote is to change the ownership of a city from one member to another (return control after conquest)
bCityVoting0Determines if the number of votes a player has is based on the number of cities they control rather than the (total\religious) population, or for religious votes the number of cities they control that have the religion.
bCivVoting0If true, then a civ can only have a maximum of 1 vote. If this is a religious vote and they don't have the religion then they don't get to vote. This takes precedence over <bCityVoting>
bDefensivePact0Whether vote is for a defensive pact between members
bForceNoTrade0Whether vote is to create an embargo against a non-member
bForcePeace0Whether vote is to end a war between members
bForceWar0Whether vote is to have members declare war against a non-member
bFreeTrade0Whether vote is for free trade between members
bNoNukes0Whether vote is to ban nukes
bOpenBorders0Whether vote is for open borders between members
bSecretaryGeneral0Whether vote is for the leader of the UN/AP
bVictory0Whether vote is for diplomatic victory
