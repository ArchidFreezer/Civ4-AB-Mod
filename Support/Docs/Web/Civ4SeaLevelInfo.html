Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
DescriptionThis declares the text tag used in the Civ4GameText_Help file to describe the effects of this option when using the "Play Now" screen on the main menu.
TypeThis is a tag used internally by the game to refer to the particular sea level choice. Prefixed by SEALEVEL_, the three default sea levels in the game are LOW, MEDIUM, and HIGH.

Integer

TagDefaultPurpose
iSeaLevelChange0This integer tag changes the percentage of water on the map by the given value. For example, SEALEVEL_LOW has a value of -8, meaning that the percentage of water on a map with the Low sea level option will be less than on a Medium sea level map, which has a value of 0.
