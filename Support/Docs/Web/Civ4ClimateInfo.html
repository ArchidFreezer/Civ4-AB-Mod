Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
DescriptionReference to the text that contains the name of the climate type. The text may be defined in any of the files in \Assets\XML\Text
TypeThe internal reference to this climate type

Integer

TagDefaultPurpose
iDesertPercentChange0Change in the percentage of desert plots that are placed
iHillRange0Factor defining the number of hills in a range
iJungleLatitude0Maximum latitude for jungle to appear
iPeakPercent0Percentage of plots that are peaks

Float

TagDefaultPurpose
fDesertBottomLatitudeChange0.0Change in the maximum latitude for desert to appear in the southern hemisphere
fDesertTopLatitudeChange0.0Change in the maximum latitude for desert to appear in the northern hemisphere
fGrassLatitudeChange0.0Change in the maximum latitude for grass to appear
fIceLatitude0.0Minumum latitude for ice
fRandIceLatitude0.0Random factor applied to the <fIceLatitude> value
fSnowLatitudeChange0.0Change in the minimum latitude for snow to appear
fTundraLatitudeChange0.0Change in the minimum latitude for tundra to appear
