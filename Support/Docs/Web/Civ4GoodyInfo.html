Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
BarbarianClassNO_UNITCLASSThe UnitClass of barbarian units that will be created
DescriptionA reference to the text files, which contain an entry for this Goody
SoundThe sound played when the goody is popped
TypeThe internal name of this Goody
UnitClassNO_UNITCLASSThe UnitClass of the unit that is gifted to the player

Integer

TagDefaultPurpose
iBarbarianUnitProb0The chance that barbarian units will be created if BarbarianClass defines a unit class
iDamagePrereq0The minimum amount of damage that a unit must have before the goody hut will consider healing them as an effect
iExperience0The experience gained by the unit popping the goody hut
iGold0Base gold received
iGoldRand10An additional random amount of gold between 0 and this value is received
iGoldRand20An additional random amount of gold between 0 and this value is received
iHealing0The percentage of a damaged unit health that is healed
iMapOffset0How far away from the goody hut the centre of the received map should be
iMapProb0Percentage chance per plot in the map range that it becomes visible
iMapRange0The radius of the map received
iMinBarbarians0If barbarians are spawned then there will be at least this number

Boolean

TagDefaultPurpose
bBad0If true, the effect is considered as bad and will not be received by certain units
bTech0If true, the player may learn a new tech
