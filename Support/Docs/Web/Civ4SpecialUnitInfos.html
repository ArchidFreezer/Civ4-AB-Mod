Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
DescriptionReference to the text that is used when the unit is being carried, e.g. "Fighters" as in Carries Fighters
TypeThe internal reference to this unit

Boolean

TagDefaultPurpose
bCityLoad0If true, the unit cannot directly rebase to a valid carrier-unit; rather it must be loaded onto carrier-units when both units are in the same city.
bValid0If false, the special unit cannot be trained globally until something unlocks it
