Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
AnyonePrereqProjectNO_PROJECTSpecifies wheter another project has already to be completed by anyone, in order to be able to build this project.
ButtonPath to a .dds file that contains the image of the project
CivilopediaReferences the Text for the pedia content. The text is specified in any of the \Assets\XML\Text files.
CreateSoundThe sound to be played to a human player when he completes the project. The possible values are defined in the Audio\Civ4Audio2DScripts.xml file
DescriptionReference to the text that contains a description of this project. The text may be defined in any of the files in \Assets\XML\Text
EveryoneSpecialBuildingNO_SPECIALBUILDINGAllows every player to build the specified type of special building. Special building types are defined in the \Assets\XML\Buildings\Civ4SpecialBuildingInfos.xml file
EveryoneSpecialUnitNO_SPECIALUNITAllows every player to train the specified type of special unit. Special unit types are defined in the \Assets\XML\UnitsCiv4SpecialUnitInfos.xml file.
MovieDefineTagNO_MOVIEThe movie to be shown to a human player when he completes the project. The possible values are defined in the \Assets\XML\Art\Civ4ArtDefines_Movie.xml file
StrategyReferences the Text for the strategy content. The text is specified in any of the \Assets\XML\Text files.
TechPrereqNO_TECHTechnology that must be known to a player before they can start on the project
TypeThe internal reference to this project
VictoryPrereqNO_VICTORYSpecifies if a victory condition has to be enabled, in order to be able to build this project.

Integer

TagDefaultPurpose
iCost0The production cost of this project. If -1, the project cannot be directly built by a city.
iMaxGlobalInstances-1Specifies how many times this project can be completed globally in the world. If -1, there's no limit.
iMaxTeamInstances-1Specifies how many times this project can be completed by a given team. If -1, there's no limit.
iNukeInterception0The chance this project has to intercept (and destroy) missiles dropped on your land.
iSuccessRate0Specifies the failure chance of the project, for each missing istance from the value of VictoryThreshold. (e.g.: if the iSuccessRate is 20 and the VictoryThreshold is 5, if I build 3 times this project I still have a 40% chance of failure; if I reach 5, the failure chance is 0).
iTechShare0Allows the project to give its owner all the techs discovered by at least this number of known civilizations.
iVictoryDelayPercent0Affects how much a victory is delayed in time, when some istances of this project have not been completed yet (see the VictoryThresholds value).

Boolean

TagDefaultPurpose
bAllowsNukes0If true, all players can now build nukes
bSpaceship0If true, this project is considered as a spaceship part for the purposes of city production bonuses and espionage missions

  <VictoryMinThreshold>

  </VictoryMinThreshold>

  </VictoryThreshold>
