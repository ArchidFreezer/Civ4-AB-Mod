Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
ColorTypeNO_COLORThis refers to a color specified in the Civ4ColorVals file that is applied to this yield type
DescriptionReference to the text that contains the name of this yield. The text may be defined in any of the files in \Assets\XML\Text
TypeThe internal reference to this yield type

Integer

TagDefaultPurpose
iAIWeightPercent0Used to control the AI's strategy and weighting of individual yields
iCityChange0Change to this yield of a plot on which a city is founded
iGoldenAgeYield0If the player enters a golden age and <iGoldenAgeYieldThreshold> is met, change the yield value by this amount.
iGoldenAgeYieldThreshold0The minimum number of yield per turn for the <iGoldenAgeYield> bonus to kick in (e.g. all tiles producing 1 or more hammers will produce an extra hammer during a golden age).
iHillsChange0Change to the yield value for plots on hills
iLakeChange0Change to the yield value for plots next to fesh water
iMinCity0The minimum yield per turn that a city can have
iPeakChange0Change to the yield value for plots on peaks
iPopulationChangeDivisor0Change to a cities yield based on the city population multiplied by the <iPopulationChangeOffset> value divided by this value
iPopulationChangeOffset0Change to a cities yield based on the city population multiplied by the this value divided by the <iPopulationChangeDivisor> value
iTradeModifier0Percentage change in the yield for each trade route in a city
