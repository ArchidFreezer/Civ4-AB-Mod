Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
AdvisorNO_ADVISORThe main focus of this technology. Used to determine which technologies your advisors will tell you to research
ButtonA comma separated field that lists four things: the path of the normal tech image, the path of the image map for this tech, and then x-y grid coordinates of the image within the image map. The path of the normal image must be written
CivilopediaRefers to a tag in the Civ4GameText file which stores the Platypedia description for the improvement
DescriptionA reference to the text files, which contain an entry for this Technology
EraNO_ERAThe pre-defined era of technologies that this technology belongs to
FirstFreeUnitClassNO_UNITCLASSProvides the first player to research this technology a unit of the specified unit class
FreeCapitalBuildingClassNO_BUILDINGCLASSProvides a player who researches this tech a building in their capital of the specified building class
FreeUnitClassNO_UNITCLASSProvides a player who researches this tech a unit of the specified unit class
HelpAdditional help text that will be displayed in the Platypedia
QuoteThe quote displayed in the civilopedia for this tech
SoundNO_SOUNDReference to the sound played when this tech is researched
SoundMPNO_SOUNDReference to the sound played when this tech is researched in a multiplayer game
StrategyThis contains the strategy text for the technology
TypeThe internal name of this Technology

Integer

TagDefaultPurpose
iAITradeModifier0Value of this technology when trading. This is used as a percentage modifier to determine the overall value of the tech.
iAIWeight0The relative value the AI puts on this technology to determine its research order
iAdvancedStartCost0Cost to start with technology via Advanced Start. All technologies have a value of 100
iAdvancedStartCostIncrease0Additional cost per technology bought through Advanced Start. All technologies have a value of 0
iAsset0The change to the value of the players assets to determine their score
iCost0The default research cost, before modifiers like difficulty level are applied.
iCultureDefenceModifier0The increase in defence that culture applies to a city
iFeatureProductionModifier0Percentage change in the production gained from worker actions, e.g. hammers gained from chopping
iFirstFreeTechs0The number of free technologies for the first leader to research this technology
iGridX0The x-coordinate, or column, to place the technology item in the tech tree
iGridY0The y-coordinate, or row, to place the technology item in the tech tree
iHappiness0Change to the happiness of each player city
iHealth0Change to the healthiness of each player city
iPower0Change to the overall power of the player as seen by the AI
iTradeRoutes0Change to the number of trade routes a city can maintain
iUnitRangeChange0Absolute change in range for a range bound unit
iUnitRangeModifier0Percentile change in range for a range bound unit
iWorkerSpeedModifier0Percentage change to the rate at which units work

Boolean

TagDefaultPurpose
bBridgeBuilding0If true, enables bridge building allowing routes to pass over rivers with no additional movement cost
bCanFoundOnPeaks0If true, settlers can found cities on peaks
bCanPassPeaks0If true, units can traverse peaks
bCaptureCities0If true, players units can capture cities
bCivSettled0xxxDESCxxx
bDefensivePactTrading0If true, allows the player to trade defensive pacts
bDisable0If true, the technology cannot be researched
bEmbassyTrading0If true, players can include the creation of an embassy in a trade deal
bExtraWaterSeeFrom0If true, increases naval units line of sight by 1
bFreeTradeAgreementTrading0If true, players can include a free trade agreement in a trade deal
bGoldTrading0If true, allows the player to trade gold
bGoodyTech0If true, the technology can be gained from a goody hut
bIgnoreIrrigation0If true, tiles are considered irrigated even if they are not next to fresh water or another irrigated tile
bIrrigation0If true, allows irrigation to be 'chained' where tiles next to irrigation can be irrigated themselves
bLimitedBordersTrading0If true, players can include a limited borders agreement in a trade deal, this agreement allows none combat units only
bMapCentering0If true, centres the world map for the player
bMapTrading0If true, allows maps to be traded in diplomacy
bMapVisible0If true, reveals the complete world map
bMoveFastPeaks0If true, units do not have an additional movement penalty when crossing peaks
bNonAggressionTrading0If true, players can include a non aggression agreement in a trade deal
bOpenBordersTrading0If true, allows the player to trade open borders
bPermanentAllianceTrading0If true, allows the player to trade permanent alliances
bRepeat0If true, allows the technology to be researched multiple times (Future Tech)
bRiverTrade0If true, allows rivers to become part of the trade network
bTechTrading0If true, allows the player to trade technologies
bTrade0If true, allows the technology to be used as part of a trade
bUnitRangeUnbound0If true, units with a maximum range have the range cap removed
bUnitTerritoryUnbound0If true, units that are normally restricted to the players territory may venture outside it
bVassalTrading0If true, allows the player to negotiate vassalage agreements, as well as capitulation during war
bWaterWork0If true, allows working of coast and ocean tiles
