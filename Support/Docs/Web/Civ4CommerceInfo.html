Tags

The tags are colour coded depending on whether they are mandatory and must be applied to all entries or whether they are optional with a default value being used if they are not present. New tags are bold.

Text

TagDefaultPurpose
ButtonPath to a .dds file that contains the image for the object
DescriptionReference to the text that containms a description of an instance of this object. The text may be defined in any of the files in \Assets\XML\Text
TypeThe internal reference to an object instance of this type

Integer

TagDefaultPurpose
iAIWeightPercent0Sets the importance of this commerce type for the AI (default is 100 for gold, 125 for research, and 25 for culture and espionage)
iInitialHappiness0Sets the happiness received by this commerce type. Each n% to that commerce type provides a happy face (default is 10 for culture, 0 for all others)
iInitialPercent0The percent value given to a commerce type when the game starts (default is 100 for research, 0 for all others).

Boolean

TagDefaultPurpose
bFlexiblePercent0Sets whether a commerce type can be given a % of your total commerce. "Flexible" commerce types have their percent slider on the main screen. "Non-Flexible" commerce types might require certain techs to be given a percent slider, like Drama for culture (default is 1 for science and espionage, 0 for gold and culture).
